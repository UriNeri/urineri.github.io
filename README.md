## Local 

1. Install the requirements:
```bash
pixi install
```

2. Serve the site locally:
```bash
pixi serve
```

open `http://127.0.0.1:8000`

## Building and Deployment

To build the site:
```bash
pixi build
```

## Deploying to GitHub Pages

To deploy to GitHub Pages:
```bash
mkdocs gh-deploy
```
